The `Icon` component is a versatile tool for adding [Expo Icons](https://icons.expo.fyi/) to your application. It offers customization options through the use of [style props](../../core-features/style-props).

## Import

```jsx
import { Icon } from "pearl-ui";
```

## Usage

```jsx
<Icon iconFamily="AntDesign" iconName="stepforward" />
```

### Icon sizes

The `size` prop allows you to adjust the size of the icon. You can use the keys available in <t>theme.components.Icon["sizes"]</t>, which are <t>"s"</t>, <t>"m"</t>, <t>"l"</t>, and <t>"xl"</t> in the [default component configuration](https://github.com/agrawal-rohit/pearl-ui/blob/main/src/components/atoms/icon/icon.config.ts).

```jsx
<Icon iconFamily="AntDesign" iconName="stepforward" size="s" />

<Icon iconFamily="AntDesign" iconName="stepforward" size="m" />

<Icon iconFamily="AntDesign" iconName="stepforward" size="l" />

<Icon iconFamily="AntDesign" iconName="stepforward" size="xl" />
```

For more precise control over the size of a single `Icon` instance, you can use the `rawSize` prop.

```jsx
<Icon iconFamily="AntDesign" iconName="stepforward" rawSize={43} />
```

### Overriding Styles

The `Icon` component supports a variety of style props which can be used to override the pre-defined variant style in the theme. Manual style props passed into the component have a higher precedence than the [default component configuration](https://github.com/agrawal-rohit/pearl-ui/blob/main/src/components/atoms/icon/icon.config.ts).

```jsx
// passing style prop color="primary.500" overrides the default component config value of color="neutral.900"
<Icon
  iconFamily="AntDesign"
  iconName="stepforward"
  size="l"
  color="primary.500"
  margin="4"
/>
```

## Accessibility

- `Icon` has the `role` of `text`.
- `Icon` has the default label set as `{iconName} Icon`. A custom label can be specified using the `accessibilityLabel` prop.

## Component Properties

### Supported Style Properties

The `Icon` component supports the following style props:

- [color and backgroundColor](../../core-features/style-props#color-and-background-color)
- [margin and padding](../../core-features/style-props#margin-and-padding)
- [layout](../../core-features/style-props#layout)
- [opacity and visibility](../../core-features/style-props#opacity-and-visibility)

### Animation Properties

The `Icon` component supports animation properties from the [Moti](https://moti.fyi/) library, enabling the creation of dynamic and interactive UI elements. The [useMotiWithStyleProps](../../hooks/useMotiWithStyleProps) hook from the Moti library is used to manage these animation properties.

| Property         | Type          | Description                                                                  |
| ---------------- | ------------- | ---------------------------------------------------------------------------- |
| `from`           | <t>object</t> | Sets the initial state of the animation.                                     |
| `to`             | <t>object</t> | Sets the final state of the animation.                                       |
| `transition`     | <t>object</t> | Defines the transition type for the animation.                               |
| `exitTransition` | <t>object</t> | Specifies the transition type when the component is unmounted.               |
| `state`          | <t>object</t> | Manages the animation's overall state, including the `from` and `to` states. |

### Additional Properties

The `Icon` component also supports the following additional properties:

| Name                 | Required | Type                                                                                                                                                                                                                                     | Default | Description                                              |
| -------------------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------- | -------------------------------------------------------- |
| `size`               | No       | <t>PearlTheme.components.Icon["sizes"]</t>                                                                                                                                                                                               | `"m"`   | Specifies the size of the icon                           |
| `variant`            | No       | <t>PearlTheme.components.Icon["variants"]</t>                                                                                                                                                                                            |         | Defines the variant of the icon                          |
| `rawSize`            | No       | <t>number</t>                                                                                                                                                                                                                            |         | Sets the raw size of the icon                            |
| `iconFamily`         | Yes      | <t> "AntDesign" \| "Entypo" \| "EvilIcons" \| "Feather" \| "FontAwesome" \| "FontAwesome5" \| "Fontisto" \| "Foundation" \| "Ionicons" \| "MaterialCommunityIcons" \| "MaterialIcons" \| "Octicons" \| "SimpleLineIcons" \| "Zocial"</t> |         | Specifies the icon family that contains the desired icon |
| `iconName`           | Yes      | <t>string</t>                                                                                                                                                                                                                            |         | Specifies the name of the icon within its icon family    |
| `accessibilityLabel` | No       | <t>string</t>                                                                                                                                                                                                                            |         | Sets the accessibility label of the icon                 |
