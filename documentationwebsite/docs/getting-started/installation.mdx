Inside your React Native/Expo project directory, install Pearl UI by running either of the following:

  values={[

  ]

<TabItem value="npm">

```bash
npm install pearl-ui
```

</TabItem>

<TabItem value="yarn">

```bash
yarn add pearl-ui
```

</TabItem>
</Tabs>
