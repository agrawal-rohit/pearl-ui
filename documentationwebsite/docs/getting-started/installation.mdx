# Installation

Inside your React Native/Expo project directory, install PearlUI by running either of the following:

<Tabs
  defaultValue="npm"
  values={[
    { label: 'NPM', value: 'npm', },
    { label: 'Yarn', value: 'yarn', },
  ]
}>
<TabItem value="npm">

```bash
npm install 'pearl-ui'
```

</TabItem>

<TabItem value="yarn">

```bash
yarn add 'pearl-ui'
```

</TabItem>
</Tabs>
