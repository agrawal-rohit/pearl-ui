The `Badge` component in Pearl UI is a small, versatile element that is typically used to display a numerical value or indicate the status of an item to the user. It can be used standalone or attached to other components to enhance their functionality.

## Import

Pearl UI provides two badge-related components for your convenience:

1. `Badge`: This is a standalone badge that can display a numerical value or an icon.
2. `withBadge`: This is a higher-order component that can be used to add a badge to any target component.

You can import these components as follows:

```jsx
import { withBadge, Badge } from "pearl-ui";
```

## Usage

```tsx
// Displaying a numerical value in the badge
<Badge>23</Badge>;

// Displaying an icon in the badge
import { Icon } from "pearl-ui";

<Badge>
  <Icon iconFamily="Entypo" iconName="edit" color="neutral.50" size="s"></Icon>
</Badge>;
```

### Badge Sizes

The `size` prop allows you to adjust the size of the badge. You can set the value to the keys available in <t>PearlTheme.components.Badge["sizes"]</t>, which are <t>"s"</t>, <t>"m"</t>, <t>"l"</t>, and <t>"xl"</t> in the [default theme](#default-component-config).

```jsx
<Badge size="s">234+</Badge>
<Badge size="m">234+</Badge>
<Badge size="l">234+</Badge>
<Badge size="xl">234+</Badge>
```

### Badge Variants

The `variant` prop allows you to change the shape of the badge. You can set the value to the keys available in <t>PearlTheme.components.Badge["variants"]</t>, which are <t>"rounded"</t> and <t>"square"</t> in the [default theme](#default-component-config).

```jsx
<Badge variant="rounded">2</Badge>
<Badge variant="square">NEW</Badge>
```

### Badge Color Scheme

The `colorScheme` prop allows you to change the color palette of the badge. You can set the value only to the keys available in <t>PearlTheme.palette</t> that contain a palette of colors represented as an object, which are <t>"primary"</t>, <t>"secondary"</t>, <t>"neutral"</t>, etc in the [default Pearl theme](../../theming/default-theme#color-palette).

```jsx
<Badge colorScheme="primary" />
<Badge colorScheme="success" />
<Badge colorScheme="warning" />
<Badge colorScheme="info" />
<Badge colorScheme="danger" />
```

### Attaching Badges to Other Components

The `Badge` component can be attached to other components using the `withBadge` function. This function allows you to add a badge to any React component in a simple manner, while also providing you the freedom to customize the badge as needed. The list of props expected by `withBadge` can be found [here](#withbadge-props).

This can be particularly useful when dealing with common use-cases of badges such as adding a notification badge as follows:

```tsx
// Attaching a badge to an Icon component
import { Icon, IconProps } from "pearl-ui";

const numberOfNotifications = 5;

const BadgedIcon = withBadge<IconProps>(numberOfNotifications, {
  placement: "topRight",
  size: "s",
  colorScheme: "danger",
})(Icon);

return <BadgedIcon iconFamily="FontAwesome" iconName="inbox" />;
```

### Overriding Styles

The `Badge` component supports a [variety of style props](#supported-style-props) which can be used to override the pre-defined variant style in the theme. Manual style props passed into the component have a higher precedence than the [default component config](#default-component-config).

```jsx
<Badge backgroundColor="blue" variant="square" mt="l" />
```

## Badge Component Properties

### Supported Style Properties

The `Badge` component composes the [Pressable](../forms/Pressable) component, so you can pass all [Pressable](../forms/Pressable) related props to it.

### Additional Properties

Here are some additional properties that the `Badge` component supports:

| Name          | Required | Type                                           | Default     | Description                        |
| ------------- | -------- | ---------------------------------------------- | ----------- | ---------------------------------- |
| `size`        | No       | <t>PearlTheme.components.Badge["sizes"]</t>    | `"m"`       | Size of the badge.                 |
| `variant`     | No       | <t>PearlTheme.components.Badge["variants"]</t> | `"rounded"` | Variant of the badge.              |
| `colorScheme` | No       | <t>PearlTheme["palette"]</t>                   | `"primary"` | Active color palette of the badge. |

## withBadge Properties

Here are the properties that the `withBadge` function supports:

| Name         | Required | Type                                                                                                                       | Default | Description                                             |
| ------------ | -------- | -------------------------------------------------------------------------------------------------------------------------- | ------- | ------------------------------------------------------- |
| `badgeValue` | No       | <t>string \| number \| React.ReactElement</t>                                                                              |         | The value to be displayed inside the badge.             |
| `hidden`     | No       | <t>boolean</t>                                                                                                             |         | Whether the badge is visible or not.                    |
| `options`    | No       | <t>BadgeProps</t> \| { placement: <t>"topLeft" \| "topRight" \| "bottomLeft" \| "bottomRight"</t>, offset: <t>number</t> } |         | A set of options for customizing the look of the badge. |
