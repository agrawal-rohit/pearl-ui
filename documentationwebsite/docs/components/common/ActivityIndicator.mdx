The `ActivityIndicator` component is the used to provide a visual cue that an action is either processing, awaiting a course of change or a result.

## Import

```jsx
import { ActivityIndicator } from "pearl-ui";
```

## Usage

```jsx
// default variant is set as "spinner" and size is set as "m" in the default Pearl theme
<ActivityIndicator />

// variant="ball" refers to the value of `theme.components["ActivityIndicator"]["variants"]["ball"]`
<ActivityIndicator variant="ball" />

// size="l" refers to the value of `theme.components["ActivityIndicator"]["sizes"]["l"]`
<ActivityIndicator variant="pacman" size="l" />

// passing style prop color="danger.500" overrides the default component style value of color="primary.500"
<ActivityIndicator color="danger.500" variant="dot" />
```

## Example

:::info
The `ActivityIndicator` component doesn't work on Expo web as the animation gets cut off after a few seconds. Please run the following example on either an Android or iOS device.
:::

import Snack from "../../../src/components/ExpoSnack";
import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

<Tabs
  defaultValue="light"
  values={[
    { label: 'Light', value: 'light', },
    { label: 'Dark', value: 'dark', },
  ]
}>
<TabItem value="light">

<Snack snackId="@agrawal-rohit/activityindicator-light" />

</TabItem>

<TabItem value="dark">

<Snack snackId="@agrawal-rohit/activityindicator-dark" />

</TabItem>
</Tabs>

## Props

### Supported style props

The `ActivityIndicator` component supports the following style props:

- [color](../../getting-started/style-props#color-and-background-color) (Only the `color` prop is supported, not the `backgroundColor` props)
- [margin and padding](../../getting-started/style-props#margin-and-padding)
- [layout](../../getting-started/style-props#layout)

### Other props

| Name    | Required | Type                                                       | Default | Description                                                                                                      |
| ------- | -------- | ---------------------------------------------------------- | ------- | ---------------------------------------------------------------------------------------------------------------- |
| size    | false    | <t>PearlTheme.components.ActivityIndicator["sizes"]</t>    | `"m"`   | ActivityIndicator size to use as defined in the active theme                                                     |
| variant | false    | <t>PearlTheme.components.ActivityIndicator["variants"]</t> | `"p3"`  | ActivityIndicator variant to use as defined in the active theme                                                  |
| loading | false    | <t>boolean</t>                                             | `true`  | The loading status of the ActivityIndicator. If `false`, the ActivityIndicator component is removed from the DOM |

## Default Component Style

```js
export default {
  baseStyle: {
    color: "primary.500",
  },
  sizes: {
    s: {
      size: 10,
    },
    m: {
      size: 20,
    },
    l: {
      size: 40,
    },
    xl: {
      size: 60,
    },
  },
  variants: {
    ball: {},
    bar: {},
    dot: {
      sizeMultiplier: 0.2,
    },
    spinner: {},
    pacman: {},
    pulse: {},
    skype: {},
    activity: {},
    wave: {},
  },
  defaults: {
    size: "m",
    variant: "spinner",
  },
};
```
