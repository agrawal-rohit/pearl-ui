The `Text` component is a versatile tool for rendering typography within your application's interface. It is designed to handle a variety of text elements, such as headings and paragraphs. By default, it renders a [React Native Text](https://reactnative.dev/docs/text) element.

## Import

```jsx
import { Text } from "pearl-ui";
```

## Usage

The `Text` component can be used to display any text within your application. By default, the variant is set as "p3" in the default Pearl theme. Here's an example:

```jsx
<Text>Hello, I am a Text component!</Text>
```

### Text variants

The `Text` component comes with a `variant` prop that allows you to change the typography style. You can set the value to the keys available in <t>PearlTheme.components.Text["variants"]</t>, which include <t>"h1"</t>, <t>"h2"</t>, and so on, as defined in the [default theme](../../theming/default-theme#typography).

```jsx
// variant="h1" refers to the value of `theme.components.Text["variants"]["h1"]`
<Text variant="h1">This is a heading</Text>
```

### Overriding Styles

The `Text` component supports a wide range of style props that can be used to override the pre-defined variant style in the theme. Any manual style props passed into the component will take precedence over the [default component config](#default-component-config).

```jsx
// variant="p1" refers to the value of `theme.components.Text["variants"]["p1"]`
// passing style prop color="primary.300" overrides the default component config value of color="neutral.50"
<Text variant="btn1" color="primary.300">
  This is a large button text
</Text>
```

## Accessibility

- `Text` has the `role` of `text`.
- `Text` has the label set as the text body.

## Component Properties

### Supported Style Properties

The `Text` component supports the following style props:

- [color and backgroundColor](../../core-features/style-props#color-and-background-color)
- [typography](../../core-features/style-props#typography)
- [layout](../../core-features/style-props#layout)
- [opacity and visiblity](../../core-features/style-props#opacity-and-visibility)
- [margin and padding](../../core-features/style-props#margin-and-padding)
- [textShadow](../../core-features/style-props#text-shadow)

### Animation Properties

The `Text` component supports animation properties from the [Moti](https://moti.fyi/) library, enabling the creation of dynamic and interactive UI elements. The [useMotiWithStyleProps](../../hooks/useMotiWithStyleProps) hook from the Moti library is used to manage these animation properties.

| Property         | Type          | Description                                                                       |
| ---------------- | ------------- | --------------------------------------------------------------------------------- |
| `from`           | <t>object</t> | Defines the initial state of the animation.                                       |
| `to`             | <t>object</t> | Defines the final state of the animation.                                         |
| `transition`     | <t>object</t> | Specifies the transition type for the animation.                                  |
| `exitTransition` | <t>object</t> | Specifies the transition type when the component is unmounted.                    |
| `state`          | <t>object</t> | Manages the overall state of the animation, including the `from` and `to` states. |

### Additional Properties

| Name            | Required | Type                                          | Default | Description                                                             |
| --------------- | -------- | --------------------------------------------- | ------- | ----------------------------------------------------------------------- |
| `size`          | No       | <t>PearlTheme.components.Text["sizes"]</t>    |         | Size variant to use as defined in the active theme.                     |
| `variant`       | No       | <t>PearlTheme.components.Text["variants"]</t> | `"p1"`  | Typography variant to use as defined in the active theme.               |
| `scaleFontSize` | No       | <t>boolean</t>                                | `true`  | Whether to slightly scale the font size based on the screen dimensions. |
