`Center` is a layout component that centers its child within itself across both axes. By default, it renders a [Box](./box) element.

## Import

```jsx
import { Center } from "pearl-ui";
```

## Usage

```jsx
<Center height={100} backgroundColor="neutral.300">
  This is the screen
</Center>
```

## Props

### Supported style props

`Center` composes [Box](./Box) so you can pass all [Box](./Box) related props.
