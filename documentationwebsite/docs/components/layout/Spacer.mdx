**Spacer** is a layout component that creates an adjustable, empty space that can be used to tune the spacing between sibling elements. By default, it renders a [Box](./box) element.

## Import

```jsx
import { Spacer } from "pearl-ui";
```

## Usage

### Horizontal Spacing

```jsx
<Box flexDirection="row">
  <Box p="l" bgColor="danger.400">
    <Text>Box 1</Text>
  </Box>
  <Spacer />
  <Box p="l" bgColor="success.400">
    <Text>Box 2</Text>
  </Box>
</Box>
```

### Vertical Spacing

```jsx
<Box flex={1}>
  <Box p="l" bgColor="danger.400">
    <Text>Box 1</Text>
  </Box>
  <Spacer />
  <Box p="l" bgColor="success.400">
    <Text>Box 2</Text>
  </Box>
</Box>
```
