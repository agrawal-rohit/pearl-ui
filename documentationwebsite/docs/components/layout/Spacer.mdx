The `Spacer` component is a flexible layout tool in Pearl UI that allows you to create adjustable, empty spaces. It is primarily used to manage and fine-tune the spacing between sibling elements in your layout. By default, it renders a [Box](./box) element.

## Import

```jsx
import { Spacer } from "pearl-ui";
```

## Usage

### Horizontal Spacing

```jsx
<Box flexDirection="row">
  <Box p="l" bgColor="danger.400">
    <Text>Box 1</Text>
  </Box>
  <Spacer />
  <Box p="l" bgColor="success.400">
    <Text>Box 2</Text>
  </Box>
</Box>
```

### Vertical Spacing

```jsx
<Box flex={1}>
  <Box p="l" bgColor="danger.400">
    <Text>Box 1</Text>
  </Box>
  <Spacer />
  <Box p="l" bgColor="success.400">
    <Text>Box 2</Text>
  </Box>
</Box>
```

## Component Properties

### Supported Style Properties

The `Spacer` component is built on top of the [Box](./Box) component, which means you can pass all [Box](./Box) related props to it.
