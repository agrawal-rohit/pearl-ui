# Box

Box is the most abstract component on top of which all other Pearl UI components are built. By default, it renders a `View` element.

The Box component is useful because it helps with creating flexible layouts with ease. It also provides an easier way to pass styles via props.

### Import

```jsx
import { Box } from "pearl-ui";
```
